# ![Logo](https://github.com/ixiaopan/EasyBarcode/raw/master/res/drawable-mdpi/ic_launcher.png) EasyBarcode

这是一个基于ZXING基础上封装的Android条码扫描类库，适合快速在项目中集成扫码功能，并且也有很强的自定义功能
